# Social Brand Scaler AI — Interactive MVP (Professional Strategist)

A polished, animated React + Vite + Tailwind frontend that simulates a brand-aware AI agent:
- Understand brand → find similar/viral posts → explain why they work → generate post drafts.
- Professional motion design: smooth transitions, subtle bubble background, typing reveal, skeletons.
- Mock backend in `src/api/mockAgent.js` so you can plug in real APIs later.

## Run
```bash
npm install
npm run dev
```
